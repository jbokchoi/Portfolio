Safe Parking LA
  Awarded the Crowd Favorite Award

Worked with non-profit Safe Parking LA to redesign their client-facing mobile site and streamline their applicaiton form for users while making results more scannable for the organization.

In this three-day social impact hackathon, working in a team of six (three developers, three UX designers), we built a responsive web-browser application to streamline processes for both website users and SPLA staff. A key priority for the team was to build a more user-friendly site (application form and better sign-posting to services), while using tools and resources the SPLA team already had access to and/or already knew how to use.

the datǝbase

A web platform designed to help singles tackle the messy world of dating. Users can store their dating contacts and track their dating activity all in one convenient place.

the datǝbase was brought together by a team of four. Git was used for version control and features were tested on branches before merging onto the master branch. I led front-end development and was responsible for overall style and UX/UI, in addition to all match-related functionality the datǝbase was built with CRUD functionality using Python on Django’s framework, HTML5, CSS3 and JavaScript to create front-end views, and PostgreSQL to store and retrieve data. Google Calendar API and Yelp API were used for additional user features and AWS to add image upload capabilities.

The Dancers' Hub

Where dancers come together to connect, unite and find opportunities. The Dancers' Hub is a digital space bringing dance professionals together to network and share latest news regarding the sector. Making it easier for artists to find resources and to help independent artists find a community to further their reach.

The Dancers' Hub is a full-stack, single-paged application incorporating the technologies of the MERN-stack (MongoDB/MongoDB Atlas/Mongoose, Express, React, Node.js). The Dancers' Hub's interactive front-end communicates with the Express backend via AJAX. Token-based authentication was used to allow for user sign-up, log in and log out.


technologies:

HTML5 | CSS3 | JavaScript | React | Node.js | Express | MongoDB | Mongoose | Git | GitHub | Bootstrap | Heroku | Adobe XD

Partners in Wine

A social catalog app that allows users to rate wines, keep track of their favorites, see what others are drinking.

The wireframes and ERD for Partners in Wine were created using Adobe XD while the project was managed with Trello. Verison control and testing was managed using Git. Partners in Wine was built with CRUD functionality using JavaScript on Node.js and Express frameworks, HTML5 and CSS3 to create front-end views, and MongoDB, MongoDB Atlas and Mongoose to store and retrieve data. User authentication was implemented using OAuth and Passport.js.


technologies:

HTML5 | CSS3 | JavaScript | Node.js | Express | MongoDB | Mongoose | OAuth | Passport.js | Git | GitHub | Heroku | Adobe XD
